жңәеҷЁеӯҰд№

еүҚиЁҖ

дј—жүҖе‘ЁзҹҘпјҢжңәеҷЁеӯҰд№ еҲҶзұ»жЁЎеһӢеёёз”ЁиҜ„д»·жҢҮж ҮжңүAccuracy, Precision, Recallе’ҢF1-scoreпјҢиҖҢеӣһеҪ’жЁЎеһӢжңҖеёёз”ЁжҢҮж ҮжңүMAEе’ҢRMSEгҖӮдҪҶжҳҜжҲ‘д»¬зңҹжӯЈдәҶи§ЈиҝҷдәӣиҜ„д»·жҢҮж Үзҡ„ж„Ҹд№үеҗ—пјҹ

еңЁе…·дҪ“еңәжҷҜпјҲеҰӮдёҚеқҮиЎЎеӨҡеҲҶзұ»пјүдёӯеҲ°еә•еә”иҜҘд»Ҙе“Әз§ҚжҢҮж Үдёәдё»иҰҒеҸӮиҖғе‘ўпјҹеӨҡеҲҶзұ»жЁЎеһӢе’ҢдәҢеҲҶзұ»жЁЎеһӢзҡ„иҜ„д»·жҢҮж Үжңүе•ҘеҢәеҲ«пјҹеӨҡеҲҶзұ»й—®йўҳдёӯпјҢдёәд»Җд№ҲAccuracy = micro precision = micro recall = micro F1-score? д»Җд№Ҳж—¶еҖҷз”Ёmacro, weighted, micro precision/ recall/ F1-score?

иҝҷеҮ еӨ©дёәдәҶеӣһеӨҚдёҘи°ЁпјҲеҲ’еҺ»: еҲҒйҡҫпјүзҡ„reviewerпјҢжҲ‘жҹҘйҳ…дәҶдёҖдәӣж–Үз« пјҢжҖ»з®—жҳҜжўізҗҶжё…жҘҡе•ҰгҖӮеңЁиҝҷйҮҢеҲҶдә«з»ҷеӨ§е®¶пјҢжқғеҪ“еҒҡдёӘжҖ»з»“гҖӮд»ҠеӨ©иҰҒи®Ізҡ„дё»иҰҒеҲҶдёәд»ҘдёӢдёӨзӮ№пјҡ

  • дәҢеҲҶзұ»жЁЎеһӢзҡ„еёёи§ҒжҢҮж Үеҝ«йҖҹеӣһйЎҫ
  • еӨҡеҲҶзұ»жЁЎеһӢзҡ„еёёи§ҒжҢҮж ҮиҜҰз»Ҷи§Јжһҗ
  • еӨҡж ҮзӯҫжЁЎеһӢзҡ„еёёи§ҒжҢҮж ҮиҜҰз»Ҷи§Јжһҗ

еңЁжҺўи®Ёиҝҷдәӣй—®йўҳеүҚпјҢи®©жҲ‘д»¬е…ҲеӣһйЎҫдёҖдёӢжңҖеёёи§Ғзҡ„жҢҮж ҮAccuracyеҲ°еә•жңүе“ӘдәӣдёҚи¶ігҖӮ

AccuracyжҳҜеҲҶзұ»й—®йўҳдёӯжңҖеёёз”Ёзҡ„жҢҮж ҮпјҢе®ғи®Ўз®—дәҶеҲҶзұ»жӯЈзЎ®зҡ„йў„жөӢж•°дёҺжҖ»йў„жөӢж•°зҡ„жҜ”еҖјгҖӮдҪҶжҳҜпјҢеҜ№дәҺдёҚе№іиЎЎж•°жҚ®йӣҶиҖҢиЁҖпјҢAccuracyе№¶дёҚжҳҜдёҖдёӘеҘҪжҢҮж ҮгҖӮдёәе•Ҙпјҹ

еҒҮи®ҫжҲ‘д»¬жңү100еј еӣҫзүҮпјҢе…¶дёӯ91еј еӣҫзүҮжҳҜгҖҢзӢ—гҖҚпјҢ5еј жҳҜгҖҢзҢ«гҖҚпјҢ4еј жҳҜгҖҢзҢӘгҖҚпјҢжҲ‘д»¬еёҢжңӣи®ӯз»ғдёҖдёӘдёүеҲҶзұ»еҷЁпјҢиғҪжӯЈзЎ®иҜҶеҲ«еӣҫзүҮйҮҢеҠЁзү©зҡ„зұ»еҲ«гҖӮе…¶дёӯпјҢзӢ—иҝҷдёӘзұ»еҲ«е°ұжҳҜеӨ§еӨҡж•°зұ» (majority class)гҖӮеҪ“еӨ§еӨҡж•°зұ»дёӯж ·жң¬пјҲзӢ—пјүзҡ„ж•°йҮҸиҝңи¶…иҝҮе…¶д»–зұ»еҲ«пјҲзҢ«гҖҒзҢӘпјүж—¶пјҢеҰӮжһңйҮҮз”ЁAccuracyжқҘиҜ„дј°еҲҶзұ»еҷЁзҡ„еҘҪеқҸпјҢйӮЈд№ҲеҚідҫҝжЁЎеһӢжҖ§иғҪеҫҲе·® (еҰӮж— и®әиҫ“е…Ҙд»Җд№ҲеӣҫзүҮпјҢйғҪйў„жөӢдёәгҖҢзӢ—гҖҚ)пјҢд№ҹеҸҜд»Ҙеҫ—еҲ°иҫғй«ҳзҡ„Accuracy ScoreпјҲеҰӮ91%пјүгҖӮжӯӨж—¶пјҢиҷҪз„¶Accuracy ScoreеҫҲй«ҳпјҢдҪҶжҳҜж„Ҹд№үдёҚеӨ§гҖӮеҪ“ж•°жҚ®ејӮеёёдёҚе№іиЎЎж—¶пјҢAccuracyиҜ„дј°ж–№жі•зҡ„зјәйҷ·е°Өдёәжҳҫи‘—гҖӮ

еӣ жӯӨпјҢжҲ‘д»¬йңҖиҰҒеј•е…ҘPrecision пјҲзІҫеҮҶеәҰпјүпјҢRecall пјҲеҸ¬еӣһзҺҮпјүе’ҢF1-scoreиҜ„дј°жҢҮж ҮгҖӮиҖғиҷ‘еҲ°дәҢеҲҶзұ»пјҢеӨҡеҲҶзұ»пјҢеӨҡж ҮзӯҫжЁЎеһӢдёӯпјҢиҜ„дј°жҢҮж Үзҡ„и®Ўз®—ж–№жі•з•ҘжңүдёҚеҗҢпјҢжҲ‘д»¬е°Ҷе…¶еҲҶејҖи®Ёи®әгҖӮ

дәҢеҲҶзұ»жЁЎеһӢзҡ„еёёи§ҒжҢҮж Үеҝ«йҖҹеӣһйЎҫ

еңЁдәҢеҲҶзұ»й—®йўҳдёӯпјҢеҒҮи®ҫиҜҘж ·жң¬дёҖе…ұжңүдёӨз§Қзұ»еҲ«пјҡPositiveе’ҢNegativeгҖӮеҪ“еҲҶзұ»еҷЁйў„жөӢз»“жқҹпјҢжҲ‘д»¬еҸҜд»Ҙз»ҳеҲ¶еҮәж··ж·Ҷзҹ©йҳөпјҲconfusion matrixпјүгҖӮе…¶дёӯеҲҶзұ»з»“жһңеҲҶдёәеҰӮдёӢеҮ з§Қпјҡ

  • True Positive (TP): жҠҠжӯЈж ·жң¬жҲҗеҠҹйў„жөӢдёәжӯЈгҖӮ
  • True Negative (TN)пјҡжҠҠиҙҹж ·жң¬жҲҗеҠҹйў„жөӢдёәиҙҹгҖӮ
  • False Positive (FP)пјҡжҠҠиҙҹж ·жң¬й”ҷиҜҜең°йў„жөӢдёәжӯЈгҖӮ
  • False Negative (FN)пјҡжҠҠжӯЈж ·жң¬й”ҷиҜҜзҡ„йў„жөӢдёәиҙҹгҖӮ

еңЁдәҢеҲҶзұ»жЁЎеһӢдёӯпјҢAccuracyпјҢPrecisionпјҢRecallе’ҢF1 scoreзҡ„е®ҡд№үеҰӮдёӢпјҡ

F1-scoreзҡ„и®Ўз®—ж–№ејҸеңЁж•°еӯҰдёӯз§°дёәпјҡи°ғе’Ңе№іеқҮгҖӮ

е…¶дёӯпјҢPrecisionзқҖйҮҚиҜ„дј°еңЁйў„жөӢдёәPositiveзҡ„жүҖжңүж•°жҚ®дёӯпјҢзңҹе®һPositveзҡ„ж•°жҚ®еҲ°еә•еҚ еӨҡе°‘пјҹRecallзқҖйҮҚиҜ„дј°пјҡеңЁжүҖжңүзҡ„Positiveж•°жҚ®дёӯпјҢеҲ°еә•жңүеӨҡе°‘ж•°жҚ®иў«жҲҗеҠҹйў„жөӢдёәPositive?

дёҫдёӘдҫӢеӯҗпјҢдёҖдёӘеҢ»йҷўж–°ејҖеҸ‘дәҶдёҖеҘ—зҷҢз—ҮAIиҜҠж–ӯзі»з»ҹпјҢжғіиҜ„дј°е…¶жҖ§иғҪеҘҪеқҸгҖӮжҲ‘д»¬жҠҠз—…дәәеҫ—дәҶзҷҢз—Үе®ҡд№үдёәPositiveпјҢжІЎеҫ—зҷҢз—Үе®ҡд№үдёәNegativeгҖӮйӮЈд№ҲпјҢ еҲ°еә•иҜҘз”Ёд»Җд№ҲжҢҮж ҮиҝӣиЎҢиҜ„дј°е‘ўпјҹ

еҰӮз”ЁPrecisionеҜ№зі»з»ҹиҝӣиЎҢиҜ„дј°пјҢйӮЈд№Ҳе…¶еӣһзӯ”зҡ„й—®йўҳе°ұжҳҜпјҡ

еңЁиҜҠж–ӯдёәзҷҢз—Үзҡ„дёҖе ҶдәәдёӯпјҢеҲ°еә•жңүеӨҡе°‘дәәзңҹеҫ—дәҶзҷҢз—Үпјҹ

еҰӮз”ЁRecallеҜ№зі»з»ҹиҝӣиЎҢиҜ„дј°пјҢйӮЈд№Ҳе…¶еӣһзӯ”зҡ„й—®йўҳе°ұжҳҜпјҡ

еңЁдёҖе Ҷеҫ—дәҶзҷҢз—Үзҡ„з—…дәәдёӯпјҢеҲ°еә•жңүеӨҡе°‘дәәиғҪиў«жҲҗеҠҹжЈҖжөӢеҮәзҷҢз—Үпјҹ

еҰӮз”ЁAccuracyеҜ№зі»з»ҹиҝӣиЎҢиҜ„дј°пјҢйӮЈд№Ҳе…¶еӣһзӯ”зҡ„й—®йўҳе°ұжҳҜпјҡ

еңЁдёҖе ҶзҷҢз—Үз—…дәәе’ҢжӯЈеёёдәәдёӯпјҢжңүеӨҡе°‘дәәиў«зі»з»ҹз»ҷеҮәдәҶжӯЈзЎ®иҜҠж–ӯз»“жһңпјҲжӮЈзҷҢжҲ–жІЎжӮЈзҷҢпјүпјҹ

OKпјҢйӮЈе•Ҙж—¶еҖҷеә”иҜҘжӣҙжіЁйҮҚRecallиҖҢдёҚжҳҜPrecisionе‘ўпјҹ

еҪ“False Negative (FN)зҡ„жҲҗжң¬д»Јд»·еҫҲй«ҳ (еҗҺжһңеҫҲдёҘйҮҚ)пјҢеёҢжңӣе°ҪйҮҸйҒҝе…Қдә§з”ҹFNж—¶пјҢеә”иҜҘзқҖйҮҚиҖғиҷ‘жҸҗй«ҳRecallжҢҮж ҮгҖӮ

еңЁдёҠиҝ°дҫӢеӯҗйҮҢпјҢFalse NegativeжҳҜеҫ—дәҶзҷҢз—Үзҡ„з—…дәәжІЎжңүиў«иҜҠж–ӯеҮәзҷҢз—ҮпјҢиҝҷз§Қжғ…еҶөжҳҜжңҖеә”иҜҘйҒҝе…Қзҡ„гҖӮжҲ‘д»¬е®ҒеҸҜжҠҠеҒҘеә·дәәиҜҜиҜҠдёәзҷҢз—Ү (FP)пјҢд№ҹдёҚиғҪи®©зңҹжӯЈжӮЈз—…зҡ„дәәжЈҖжөӢдёҚеҮәзҷҢз—Ү (FN) иҖҢиҖҪиҜҜжІ»з–—зҰ»дё–гҖӮеңЁиҝҷйҮҢпјҢзҷҢз—ҮиҜҠж–ӯзі»з»ҹзҡ„зӣ®ж ҮжҳҜпјҡе°ҪеҸҜиғҪжҸҗй«ҳRecallеҖјпјҢе“ӘжҖ•зүәзүІдёҖйғЁеҲҶPrecisionгҖӮ

йӮЈе•Ҙж—¶еҖҷеә”иҜҘжӣҙжіЁйҮҚPrecisionиҖҢдёҚжҳҜRecallе‘ўпјҹ

еҪ“False Positive (FP)зҡ„жҲҗжң¬д»Јд»·еҫҲй«ҳ (еҗҺжһңеҫҲдёҘйҮҚ)ж—¶пјҢеҚіжңҹжңӣе°ҪйҮҸйҒҝе…Қдә§з”ҹFPж—¶пјҢеә”иҜҘзқҖйҮҚиҖғиҷ‘жҸҗй«ҳPrecisionжҢҮж ҮгҖӮ

д»ҘеһғеңҫйӮ®д»¶еұҸи”Ҫзі»з»ҹдёәдҫӢпјҢеһғеңҫйӮ®д»¶дёәPositiveпјҢжӯЈеёёйӮ®д»¶дёәNegativeпјҢFalse PositiveжҳҜжҠҠжӯЈеёёйӮ®д»¶иҜҶеҲ«дёәеһғеңҫйӮ®д»¶пјҢиҝҷз§Қжғ…еҶөжҳҜжңҖеә”иҜҘйҒҝе…Қзҡ„пјҲдҪ иғҪе®№еҝҚдёҖе°ҒйҮҚиҰҒе·ҘдҪңйӮ®д»¶зӣҙжҺҘиҝӣдәҶеһғеңҫз®ұпјҢиў«дёҚзҹҘдёҚи§үеҲ йҷӨеҗ—пјҹпјүгҖӮжҲ‘д»¬е®ҒеҸҜжҠҠеһғеңҫйӮ®д»¶ж Үи®°дёәжӯЈеёёйӮ®д»¶ (FN)пјҢд№ҹдёҚиғҪи®©жӯЈеёёйӮ®д»¶зӣҙжҺҘиҝӣеһғеңҫз®ұ (FP)гҖӮеңЁиҝҷйҮҢпјҢеһғеңҫйӮ®д»¶еұҸи”Ҫзі»з»ҹзҡ„зӣ®ж ҮжҳҜпјҡе°ҪеҸҜиғҪжҸҗй«ҳPrecisionеҖјпјҢе“ӘжҖ•зүәзүІдёҖйғЁеҲҶrecallгҖӮ

иҖҢF1-scoreжҳҜPrecisionе’ҢRecallдёӨиҖ…зҡ„з»јеҗҲгҖӮ

дёҫдёӘжӣҙжңүж„ҸжҖқзҡ„дҫӢеӯҗпјҲжҲ‘жӢҚи„‘иўӢжғіеҮәжқҘзҡ„пјҢз»қеҜ№еҺҹеҲӣе“ҲпјүпјҢеҒҮи®ҫжЈҖеҜҹжңәе…іжғіе°ҶзҪӘзҠҜжҚүжӢҝеҪ’жЎҲпјҢйңҖиҰҒеҜ№жүҖжңүдәәзҫӨиҝӣиЎҢеҲҶжһҗпјҢд»ҘеҲӨж–ӯжҹҗдәәзҠҜдәҶзҪӘпјҲPositiveпјүпјҢиҝҳжҳҜжІЎзҠҜзҪӘпјҲNegativeпјүгҖӮжҳҫз„¶пјҢжЈҖеҜҹжңәе…іеёҢжңӣдёҚжјҸжҺүдёҖдёӘзҪӘдәәпјҲжҸҗй«ҳrecallпјүпјҢд№ҹдёҚй”ҷжҖӘдёҖдёӘеҘҪдәәпјҲжҸҗй«ҳprecisionпјүпјҢжүҖд»Ҙе°ұйңҖиҰҒеҗҢж—¶жқғиЎЎrecallе’ҢprecisionдёӨдёӘжҢҮж ҮгҖӮ

е°Өе…¶еңЁдёҠдёӘдё–зәӘпјҢдёӯеӣҪеҸёжі•дҪ“еҲ¶дјҡжӣҙеҒҸеҗ‘RecallпјҢеҚігҖҢеӨ©зҪ‘жҒўжҒўпјҢз–ҸиҖҢдёҚжјҸпјҢд»»дҪ•зҪӘзҠҜйғҪжҸ’зҝ…йҡҫйЈһгҖҚгҖӮиҖҢиҘҝж–№еҸёжі•зі»з»ҹдјҡжӣҙеҒҸеҗ‘PrecisionпјҢеҚігҖҢз»қдёҚеҶӨжһүдёҖдёӘеҘҪдәәпјҢдҪҶжҳҜйҡҫе…ҚжңүзҪӘзҠҜжҲҗдёәжјҸзҪ‘д№ӢйұјпјҢйҖҚйҒҘжі•еӨ–гҖҚгҖӮеҲ°еә•жҳҜе“Әз§ҚжӣҙеҘҪе‘ўпјҹжҳҫз„¶пјҢжһҒз«Ҝе№¶дёҚеҸҜеҸ–гҖӮPrecisionе’ҢRecallйғҪеә”иҜҘи¶Ҡй«ҳи¶ҠеҘҪпјҢд№ҹе°ұжҳҜF1еә”иҜҘи¶Ҡй«ҳи¶ҠеҘҪгҖӮ

е‘јпјҢдәҢеҲҶзұ»й—®йўҳзҡ„еёёи§ҒжҢҮж Үе’ҢиҜ•з”ЁеңәжҷҜз»ҲдәҺи®Іе®ҢдәҶгҖӮе’ҰпјҢиҜҙеҘҪзҡ„еҝ«йҖҹеӣһйЎҫе‘ўпјҹ

еӨҡеҲҶзұ»жЁЎеһӢзҡ„еёёи§ҒжҢҮж ҮиҜҰз»Ҷи§Јжһҗ

еңЁеӨҡеҲҶзұ»пјҲеӨ§дәҺдёӨдёӘзұ»пјүй—®йўҳдёӯпјҢеҒҮи®ҫжҲ‘д»¬иҰҒејҖеҸ‘дёҖдёӘеҠЁзү©иҜҶеҲ«зі»з»ҹпјҢжқҘеҢәеҲҶиҫ“е…ҘеӣҫзүҮжҳҜзҢ«пјҢзӢ—иҝҳжҳҜзҢӘгҖӮз»ҷе®ҡеҲҶзұ»еҷЁдёҖе ҶеҠЁзү©еӣҫзүҮпјҢдә§з”ҹдәҶеҰӮдёӢз»“жһңж··ж·Ҷзҹ©йҳөгҖӮ

еңЁж··ж·Ҷзҹ©йҳөдёӯпјҢжӯЈзЎ®зҡ„еҲҶзұ»ж ·жң¬пјҲActual label = Predicted labelпјүеҲҶеёғеңЁе·ҰдёҠеҲ°еҸідёӢзҡ„еҜ№и§’зәҝдёҠгҖӮе…¶дёӯпјҢAccuracyзҡ„е®ҡд№үдёәеҲҶзұ»жӯЈзЎ®пјҲеҜ№и§’зәҝдёҠпјүзҡ„ж ·жң¬ж•°дёҺжҖ»ж ·жң¬ж•°зҡ„жҜ”еҖјгҖӮAccuracyеәҰйҮҸзҡ„жҳҜе…ЁеұҖж ·жң¬йў„жөӢжғ…еҶөгҖӮиҖҢеҜ№дәҺPrecisionе’ҢRecallиҖҢиЁҖпјҢжҜҸдёӘзұ»йғҪйңҖиҰҒеҚ•зӢ¬и®Ўз®—е…¶Precisionе’ҢRecallгҖӮ

жҜ”еҰӮпјҢеҜ№зұ»еҲ«гҖҢзҢӘгҖҚиҖҢиЁҖпјҢе…¶Precisionе’ҢRecallеҲҶеҲ«дёә:

д№ҹе°ұжҳҜпјҢ

пјҲPд»ЈиЎЁPrecisionпјү

пјҲRд»ЈиЎЁRecallпјү

еҰӮжһңжғіиҜ„дј°иҜҘиҜҶеҲ«зі»з»ҹзҡ„жҖ»дҪ“еҠҹиғҪпјҢеҝ…йЎ»иҖғиҷ‘зҢ«гҖҒзӢ—гҖҒзҢӘдёүдёӘзұ»еҲ«зҡ„з»јеҗҲйў„жөӢжҖ§иғҪгҖӮйӮЈд№ҲпјҢеҲ°еә•иҰҒжҖҺд№Ҳз»јеҗҲиҝҷдёүдёӘзұ»еҲ«зҡ„Precisionе‘ўпјҹжҳҜз®ҖеҚ•еҠ иө·жқҘеҒҡе№іеқҮеҗ—пјҹйҖҡеёёжқҘиҜҙпјҢ жҲ‘д»¬жңүеҰӮдёӢеҮ з§Қи§ЈеҶіж–№жЎҲпјҲд№ҹеҸҜеҸӮиҖғscikit-learnе®ҳзҪ‘пјүпјҡ

1. Macro-averageж–№жі•

иҜҘж–№жі•жңҖз®ҖеҚ•пјҢзӣҙжҺҘе°ҶдёҚеҗҢзұ»еҲ«зҡ„иҜ„дј°жҢҮж ҮпјҲPrecision/ Recall/ F1-scoreпјүеҠ иө·жқҘжұӮе№іеқҮпјҢз»ҷжүҖжңүзұ»еҲ«зӣёеҗҢзҡ„жқғйҮҚгҖӮиҜҘж–№жі•иғҪеӨҹе№ізӯүзңӢеҫ…жҜҸдёӘзұ»еҲ«пјҢдҪҶжҳҜе®ғзҡ„еҖјдјҡеҸ—зЁҖжңүзұ»еҲ«еҪұе“ҚгҖӮ

2. Weighted-averageж–№жі•

иҜҘж–№жі•з»ҷдёҚеҗҢзұ»еҲ«дёҚеҗҢжқғйҮҚпјҲжқғйҮҚж №жҚ®иҜҘзұ»еҲ«зҡ„зңҹе®һеҲҶеёғжҜ”дҫӢзЎ®е®ҡпјүпјҢжҜҸдёӘзұ»еҲ«д№ҳжқғйҮҚеҗҺеҶҚиҝӣиЎҢзӣёеҠ гҖӮиҜҘж–№жі•иҖғиҷ‘дәҶзұ»еҲ«дёҚе№іиЎЎжғ…еҶөпјҢе®ғзҡ„еҖјжӣҙе®№жҳ“еҸ—еҲ°еёёи§Ғзұ»пјҲmajority classпјүзҡ„еҪұе“ҚгҖӮ

(Wд»ЈиЎЁжқғйҮҚпјҢNд»ЈиЎЁж ·жң¬еңЁиҜҘзұ»еҲ«дёӢзҡ„зңҹе®һж•°зӣ®)

3. Micro-averageж–№жі•

иҜҘж–№жі•жҠҠжҜҸдёӘзұ»еҲ«зҡ„TP, FP, FNе…ҲзӣёеҠ д№ӢеҗҺпјҢеңЁж №жҚ®дәҢеҲҶзұ»зҡ„е…¬ејҸиҝӣиЎҢи®Ўз®—гҖӮ

е…¶дёӯпјҢзү№еҲ«жңүж„ҸжҖқзҡ„жҳҜпјҢMicro-precisionе’ҢMicro-recallз«ҹз„¶е§Ӣз»ҲзӣёеҗҢпјҒиҝҷжҳҜдёәе•Ҙе‘ўпјҹ

иҝҷжҳҜеӣ дёәеңЁжҹҗдёҖзұ»дёӯзҡ„False Positiveж ·жң¬пјҢдёҖе®ҡжҳҜе…¶д»–жҹҗзұ»еҲ«зҡ„False Negativeж ·жң¬гҖӮеҗ¬иө·жқҘжңүзӮ№жҠҪиұЎпјҹдёҫдёӘдҫӢеӯҗпјҢжҜ”еҰӮиҜҙзі»з»ҹй”ҷжҠҠгҖҢзӢ—гҖҚйў„жөӢжҲҗгҖҢзҢ«гҖҚпјҢйӮЈд№ҲеҜ№дәҺзӢ—иҖҢиЁҖпјҢе…¶й”ҷиҜҜзұ»еһӢе°ұжҳҜFalse NegativeпјҢеҜ№дәҺзҢ«иҖҢиЁҖпјҢе…¶й”ҷиҜҜзұ»еһӢе°ұжҳҜFalse PositiveгҖӮдәҺжӯӨеҗҢж—¶пјҢMicro-precisionе’ҢMicro-recallзҡ„ж•°еҖјйғҪзӯүдәҺAccuracyпјҢеӣ дёәе®ғд»¬и®Ўз®—дәҶеҜ№и§’зәҝж ·жң¬ж•°е’ҢжҖ»ж ·жң¬ж•°зҡ„жҜ”еҖјпјҢжҖ»з»“е°ұжҳҜпјҡ

жңҖеҗҺпјҢжҲ‘д»¬иҝҗиЎҢдёҖдёӢд»Јз ҒпјҢжЈҖйӘҢжүӢеҠЁи®Ўз®—з»“жһңжҳҜеҗҰе’ҢSklearnеҢ…з»“жһңдёҖиҮҙпјҡ

import numpy as np
import seaborn as sns
from sklearn.metrics import confusion_matrix
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.metrics import accuracy_score, average_precision_score,precision_score,f1_score,recall_score

# create confusion matrix
y_true = np.array([-1]*70 + [0]*160 + [1]*30)
y_pred = np.array([-1]*40 + [0]*20 + [1]*20 + 
                  [-1]*30 + [0]*80 + [1]*30 + 
                  [-1]*5 + [0]*15 + [1]*20)
cm = confusion_matrix(y_true, y_pred)
conf_matrix = pd.DataFrame(cm, index=['Cat','Dog','Pig'], columns=['Cat','Dog','Pig'])

# plot size setting
fig, ax = plt.subplots(figsize = (4.5,3.5))
sns.heatmap(conf_matrix, annot=True, annot_kws={"size": 19}, cmap="Blues")
plt.ylabel('True label', fontsize=18)
plt.xlabel('Predicted label', fontsize=18)
plt.xticks(fontsize=18)
plt.yticks(fontsize=18)
plt.savefig('confusion.pdf', bbox_inches='tight')
plt.show()

print('-----Macro-----')
print('Macro precision',precision_score(y_true,y_pred,average='macro'))
print('Macro recall',recall_score(y_true,y_pred,average='macro'))
print('Macro f1-score',f1_score(y_true,y_pred,average='macro'))

print('-----Weighted-----')
print('Weighted precision',precision_score(y_true,y_pred,average='weighted'))
print('Weighted recall',recall_score(y_true,y_pred,average='weighted'))
print('Weighted f1-score',f1_score(y_true,y_pred,average='weighted'))

print('-----Micro-----')
print('Micro precision',precision_score(y_true,y_pred,average='micro'))
print('Micro recall',recall_score(y_true,y_pred,average='micro'))
print('Micro f1-score',f1_score(y_true,y_pred,average='micro'))

иҝҗз®—з»“жһңе®Ңе…ЁдёҖиҮҙгҖӮ

еӨҡж ҮзӯҫжЁЎеһӢзҡ„еёёи§ҒжҢҮж ҮиҜҰз»Ҷи§Јжһҗ

еүҚйқўи®Іи§ЈдәҶдәҢеҲҶзұ»е’ҢеӨҡеҲҶзұ»жЁЎеһӢзҡ„иҜ„д»·жҢҮж ҮпјҢйӮЈд№ҲеҜ№дәҺеӨҡж ҮзӯҫеҲҶзұ»жЁЎеһӢпјҢе®ғзҡ„иҜ„д»·жҢҮж ҮеҸҲжҳҜжҖҺд№Ҳи®Ўз®—зҡ„е‘ўпјҹ

еңЁй’ҲеҜ№multilabelеҲҶзұ»и®Ўз®—F scoreзҡ„ж—¶еҖҷпјҢйҖҡеёёжңүmacroе’ҢmicroдёӨз§Қaverageзҡ„ж–№жі•гҖӮPythonзҡ„scikit-learnеә“еңЁи®Ўз®—f1 scoreд№ҹжҸҗдҫӣдәҶmicroе’ҢmacroдёӨз§ҚйҖүжӢ©пјҢе…·дҪ“еңЁmultilabelзҡ„жғ…еҶөдёӢпјҢжҖҺд№Ҳи®Ўз®—F1-scoreпјҢеңЁзҪ‘дёҠжҹҘйҳ…дәҶеҫҲеӨҡеҚҡе®ўе’Ңиө„ж–ҷйғҪжІЎжңүз»ҷеҮәдёҖдёӘжҳҺзЎ®зҡ„з”ЁеҲ—еӯҗи§ЈйҮҠзҡ„жӯҘйӘӨпјҢиҝҷиҫ№жҲ‘иҮӘе·ұйҖҡиҝҮж•ҙеҗҲиө„ж–ҷд»Јз ҒйӘҢиҜҒеҮәдәҶmacroе’ҢmicroдёӨз§ҚF1-scoreзҡ„и®Ўз®—ж–№жі•гҖӮ

иҜ·зңӢдёӢйқўзҡ„з®ҖеҚ•дҫӢеӯҗпјҡ

и®©$N=3,L=3$пјҢдёҖе…ұжңү3дёӘdataпјҢжҜҸдёӘdataжңү3дёӘйў„еӨҮеҲҶзұ»гҖӮ

y_gt = np.array([[1,0,1],[0,1,1],[0,1,0]])
y_pred = np.array([[0,0,1],[1,1,1],[1,1,1]])

еҜ№дәҺжҜҸдёҖдёӘclassпјҢжҲ‘д»¬йғҪйңҖиҰҒе…Ҳз®—дёҖдёӘ2x2зҡ„confusion matrixпјҢйҮҢйқўеҲҶеҲ«ж ҮжҳҺдәҶеҜ№дәҺиҝҷдёҖдёӘclassзҡ„TPпјҢFPпјҢFNе’ҢTNгҖӮе…¶е®һеҒҡжі•е’ҢеӨҡеҲҶзұ»жҳҜдёҖж ·зҡ„гҖӮ

1. Macro-averageж–№жі•

еҜ№дәҺmacroпјҢжҲ‘д»¬йҖҡиҝҮжҜҸдёҖдёӘclassзҡ„confusion matrixз®—еҮәе®ғзҡ„precisionе’ҢrecallпјҢе№¶и®Ўз®—еҮәеҜ№дёҺйӮЈдёӘclassзҡ„F1 scoreпјҢжңҖеҗҺйҖҡиҝҮе№іеқҮжүҖжңүclassзҡ„F1 scoreеҫ—еҲ°F1 macro

Class 0пјҡ

gt\pred 1 0
1 0 1
0 2 0

$P_0 = \frac{0}{0+2}=0, R_0 = \frac{0}{0+1}=0$

$F1_0=\frac{2\times0\times0}{0+0}=0$

Class 1пјҡ

gt\pred 1 0
1 2 0
0 0 1

$P_1 = \frac{2}{2+0}=1, R_1 = \frac{2}{2+0}=1$

$F1_1=\frac{2\times1\times1}{1+1}=1$

Class 2:

gt\pred 1 0
1 2 0
0 1 0

$P_2 = \frac{2}{2+1}=\frac{2}{3}, R_2 = \frac{2}{2+0}=1$

$F1_2=\frac{2\times{\frac{2}{3}}\times1}{\frac{2}{3}+1}=\frac{4}{5}$

$F1_{macro}=\frac{1}{N}(F1_0+ F1_1+вҖҰ+F1_N)=\frac{1}{3}(F1_0+F1_1+F1_2)=\frac{1}{3}(0+1+\frac{4}{5})=0.6$

2. Micro-averageж–№жі•

еҜ№дәҺmicroпјҢжҲ‘д»¬жҠҠжүҖжңүclassзҡ„binary confusion matrixж•ҙеҗҲжҲҗдёҖдёӘеӨ§зҡ„2x2confusion matrixпјҢз„¶еҗҺе№¶еҜ№дәҺж•ҙеҗҲжҲҗзҡ„confusion matrixз®—еҮәдёҖдёӘprecisionе’ҢrecallеҖјпјҲ$P_{comb}$ and $R_{comb}$пјүпјҢжңҖеҗҺйҖҡиҝҮе…¬ејҸеҫ—еҲ°$F1_{micro}$ scoreгҖӮ

Combined all classes:

gt\pred 1 0
1 4 1
0 3 1

$P_{comb}=\frac{4}{4+3}=\frac{4}{7}, R_{comb}=\frac{4}{4+1}=\frac{4}{5}$

$F1_{micro}=\frac{2P_{comb}R_{comb}}{P_{comb}+R_{comb}}=\frac{2\times\frac{4}{7}\times\frac{4}{5}}{\frac{4}{7}+\frac{4}{5}}=0.66666666вҖҰ$

жіЁж„ҸеҢәеҲ«еңЁеӨҡж ҮзӯҫжЁЎеһӢдёӯпјҢе…¶Micro-averageеҫ—еҲ°зҡ„precisionпјҢrecallе’ҢF1-scoreе№¶дёҚжҳҜзӣёзӯүзҡ„гҖӮ
